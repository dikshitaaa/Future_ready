## 📌 Overview

**FutureReady** is a comprehensive AI-powered career guidance platform that helps students and professionals discover their ideal career paths, access detailed roadmaps, practice interview questions, and track their progress - all in one place.

🔗 **Live Demo:** [View Website](https://dikshitaaa.github.io/Future_ready/)

---

###  Core Features

| Feature | Description |
|---------|-------------|
| **Career Quiz** | 10-question quiz to discover your ideal career |
| **70+ Careers** | Comprehensive career database with detailed roadmaps |
| **Skill Roadmap** | Step-by-step learning paths for each career |
| **AI Counselor** | Chatbot for personalized career advice |
| **Jobs in Demand** | Salary insights and industry trends |
| **Career Bingo** | Gamified task completion with badges |
| **Resume Builder** | Create and download professional resumes as PDF |
| **Daily Challenge** | Daily quizzes to test knowledge |
| **Notes Diary** | Personal notes with auto-save feature |
| **Interview Questions** | 1000+ career-specific questions |
| **Dark Mode** | Eye-friendly dark theme |
| **Responsive Design** | Works on all devices |
